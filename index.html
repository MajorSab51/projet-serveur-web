Présentation du Raspberry Pi

  • Le Raspberry Pi est un mini-ordinateur, créé pour rendre l’informatique accessible à tous. Il a été développé par la Raspberry Pi Foundation,la première version en 2012.

  • Il peut être relié à un écran, un clavier et une souris, et fonctionne avec un système d’exploitation basé sur Linux.

  • Le Raspberry Pi est principalement utilisé pour apprendre la programmation, notamment avec des langages.

  • Il permet aussi de réaliser de nombreux projets, comme la robotique, la domotique, ou la création d’un serveur web. Ses principaux avantages sont son faible coût, sa simplicité d’utilisation et sa grande polyvalence.

Raspberry Pi OS

  • Raspberry Pi OS est le système d'exploitation spécialement développé pour Raspberry Pi basé sur Debian.

HTML, CSS & Javascript

  • Le HTML (HyperText Markup Language), ou « langage de balisage hypertexte », est le langage fondamental utilisé pour créer des pages web. Il permet de structurer et d’organiser le contenu affiché sur Internet.

  • Le HTML fonctionne grâce à des balises, qui encadrent les différents éléments d’une page. Ces balises indiquent au navigateur comment interpréter le contenu. Par exemple, certaines balises servent à définir des titres, des paragraphes, des images ou encore des liens.

  • le HTML constitue la base de tout site web. Il est indispensable pour organiser les informations et permettre leur affichage correct dans un navigateur.
