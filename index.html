Introduction

  • Utilisation du Raspberry Pi comme serveur web.

  • Objectif : comprendre le fonctionnement d'un serveur web et des échanges réseau.

  • Cette page web réalisé par l'équipe et hébergé sur le Raspberry Pi sert de support à la présentation.

Raspberry Pi

  • Nano-ordinateur : contient un processeur, de la mémoire vive et du stockage.

  • Présence de ports : USB, HDMI, Ethernet, USB-C.

  • Petit, faible consommation électrique, faible coût.

  • Accessible, polyvalent, peut fonctionner en continu. Permet d'apprendre Linux, le développement et pour créer des projets électroniques.

Raspberry Pi OS

  • Basé sur Debian

Apache HTTP Server

  • Serveur web le plus utilisé dans le monde.

  • Reçoit les requêtes HTTP, traite les demandes et envoie la page web.

  • Facile d'utilisation et adapté aux débutants.

HTML & CSS

  • HTML : utilise des balises pour ajouter les éléments et structurer la page

  • CSS : permet de styliser une page web (couleur, taille, disposition et alignement des éléments.)

  • Fichiers envoyé à la machine client.

Sécurité

Essentielle car serveur accessible depuis Internet

Risques :

  • Failles :
    • Attaques / tentatives d'intrusion.
    • Accès non autorisé aux fichiers.
    • Modification du site.
  • Protection :
    • Utilisation d'un pare-feu.
    • Utilisation de HTTPS.
    • Mettre à jour les logiciels régulièrement.
    • Gestion des droits Linux

Applications concrètes

  • Héberger un site personnel
  • Gérer un serveur de fichier
  • Gérer la domotique d'une maison
  • Tester des projets web
  • Gagner des compétences en réseaux et serveurs.
